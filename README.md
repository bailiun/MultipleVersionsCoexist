MultipleVersionsCoexist

MultipleVersionsCoexist 是一个面向中大型业务场景的 接口级多版本共存管理框架。它基于 Spring/Spring Boot 深度扩展，提供一套可控、低侵入的服务版本化解决方案，让同一项目中多个 API 版本、多个实现、多个语言模块能够安全共存并灵活调度。

本组件核心能力包括：

接口级多版本共存（Version Coexist）
支持同一接口的多版本实现同时运行，可根据请求、环境、策略进行精确路由，让 Beta 测试、A/B 实验、MVP 验证变得极简。

可控的 Bean/接口注入与访问流程
可自定义优先级、环境规则、注入策略，实现“按需加载、按环境隔离”的运行模型。

独特的反射式 AOP 切面模型
无需繁琐的代理配置，即可使用简洁 API 完成切面逻辑，实现动态增强、行为注入、轻量化埋点等能力。

多语言兼容
内置 Python 脚本执行与交互支持，未来将扩展至更多语言，助力跨语言微服务或混合开发场景。

MultipleVersionsCoexist 的目标是：
让单个开发者也能轻松驾驭原本只有大型团队才会使用的复杂架构能力。
